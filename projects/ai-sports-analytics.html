Capstone Project

AI Sports Analytics System

Computer vision and machine learning platform for athletic performance analysis

Spring 2025 Python, TensorFlow, OpenCV 3-month development

Project Overview

Goal

Develop an AI-powered system that uses computer vision and machine learning to analyze player performance, predict game outcomes, and provide actionable insights for coaches and teams.

Why I Built This

Sports analytics is rapidly evolving, and AI can provide deeper insights than traditional statistics. I wanted to combine my passion for sports with my technical skills in AI and data analysis to create something with practical applications.

Key Challenge

Processing real-time video data from multiple camera angles and accurately tracking player movements in complex, fast-paced game environments.

Technical Implementation

System Architecture

Data Collection Layer

Multi-angle video feeds → OpenCV processing → Frame extraction

Processing Layer

YOLOv8 object detection → Player tracking → Feature extraction

AI/ML Layer

TensorFlow models → Performance prediction → Pattern recognition

Presentation Layer

Flask web interface → Real-time dashboards → Coach notifications

Key Features Developed

Player Movement Tracking

Used OpenCV and YOLOv8 to detect and track players across video frames, calculating speed, distance, and positioning metrics.

Performance Prediction

Trained LSTM neural networks on historical data to predict individual and team performance metrics.

Real-time Dashboard

Built a Flask web application with real-time data visualization using Plotly and D3.js.

Player Detection with YOLOv8

import cv2
import torch
from ultralytics import YOLO

class PlayerTracker:
    def __init__(self, model_path='yolov8n.pt'):
        self.model = YOLO(model_path)
        self.track_history = []
    
    def track_players(self, video_path):
        """Process video and track player movements"""
        cap = cv2.VideoCapture(video_path)
        player_positions = []
        
        while cap.isOpened():
            success, frame = cap.read()
            if not success:
                break
                
            # Run YOLOv8 tracking
            results = self.model.track(frame, persist=True)
            
            # Extract player positions
            if results[0].boxes.id is not None:
                boxes = results[0].boxes.xywh.cpu()
                track_ids = results[0].boxes.id.int().cpu().tolist()
                
                for box, track_id in zip(boxes, track_ids):
                    x, y, w, h = box
                    player_positions.append({
                        'frame': frame_count,
                        'player_id': track_id,
                        'position': (float(x), float(y)),
                        'velocity': self.calculate_velocity(track_id, x, y)
                    })
            
            # Update tracking history
            self.update_tracking_history(player_positions)
            
        cap.release()
        return self.analyze_movements(player_positions)

Performance Prediction Model

import tensorflow as tf
from tensorflow.keras.models import Sequential
from tensorflow.keras.layers import LSTM, Dense, Dropout
import numpy as np

class PerformancePredictor:
    def __init__(self):
        self.model = self.build_model()
    
    def build_model(self):
        """Build LSTM model for performance prediction"""
        model = Sequential([
            LSTM(128, return_sequences=True, input_shape=(10, 15)),
            Dropout(0.3),
            LSTM(64, return_sequences=False),
            Dropout(0.3),
            Dense(32, activation='relu'),
            Dense(1, activation='linear')  # Predict performance score
        ])
        
        model.compile(
            optimizer='adam',
            loss='mse',
            metrics=['mae']
        )
        return model
    
    def prepare_data(self, player_data):
        """Prepare sequential data for LSTM"""
        sequences = []
        targets = []
        
        for i in range(len(player_data) - 10):
            sequence = player_data[i:i+10]
            target = player_data[i+10]
            sequences.append(sequence)
            targets.append(target)
        
        return np.array(sequences), np.array(targets)
